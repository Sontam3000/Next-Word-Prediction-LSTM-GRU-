# Next-Word-Prediction-LSTM-GRU-
#Next Word Prediction App (LSTM & GRU)

This project is a real-time Next Word Prediction web application built using
LSTM and GRU neural networks trained on Shakespeare's Hamlet corpus.

## Features
- Live word prediction (no button click)
- Dropdown to switch between LSTM and GRU models
- Top-k word suggestions with probabilities
- Built using TensorFlow and Streamlit

## Tech Stack
- Python
- TensorFlow (LSTM, GRU)
- Streamlit
